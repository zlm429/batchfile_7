# 📋 剪贴板方式操作指南

## ✨ 全新操作方式

现在您可以通过**Windows资源管理器直接选中文件夹**，然后让程序获取这些文件夹进行批量重命名！

---

## 🚀 操作步骤

### 第一步：准备Excel文件

创建Excel文件，包含4列数据（A、B、C、D）：

```
| A列            | B列    | C列           | D列            |
|---------------|-------|--------------|---------------|
| J76000016.3426 | 雾卡其 | 170/92A(M)   | 6942674037033 |
| J76000016.3427 | 雾卡其 | 175/96A(L)   | 6942674037040 |
| J76000016.3428 | 雾卡其 | 180/100A(XL) | 6942674037057 |
```

程序会自动将4列组合成：`J76000016.3426_雾卡其_170/92A(M)_6942674037033`

**注意**：Excel中的 `/` 等特殊字符会被自动转换为 `_`

### 第二步：在资源管理器中选中文件夹

1. 打开Windows资源管理器
2. 导航到包含需要重命名文件夹的目录
3. **选中需要重命名的文件夹**（可以多选）：
   - 按住 `Ctrl` 点击选择不连续的文件夹
   - 按住 `Shift` 点击选择连续的文件夹
   - 或用鼠标拖拽框选

4. 按 `Ctrl + C` 复制选中的文件夹

### 第三步：启动程序

```powershell
# 双击运行
run.bat

# 或使用PowerShell
.\run.ps1

# 或使用Maven
.\mvnw.cmd clean javafx:run
```

### 第四步：加载Excel数据

1. 点击界面上的"浏览..."按钮
2. 选择准备好的Excel文件
3. 日志显示："从Excel中加载了 X 个名称（4列组合）"

### 第五步：从剪贴板获取文件夹

1. 点击界面上的 **"📋 从剪贴板获取文件夹"** 按钮
2. 程序会自动从剪贴板读取您刚才复制的文件夹
3. 蓝色文本框会显示所有选中文件夹的完整路径
4. 日志显示："从剪贴板获取了 X 个文件夹"

### 第六步：执行重命名

1. 确认选中的文件夹数量正确
2. 点击 **"开始重命名"** 按钮
3. 观察日志区域的详细操作过程
4. 完成后会显示成功和失败的数量

## ⚠️ 重要提醒

### 1. 日志乱码问题已修复
- 程序已设置UTF-8编码
- 日志区域使用Consolas字体
- 如果仍有乱码，请确保系统支持中文显示

### 2. 重命名失败常见原因

**原因1：文件夹被占用**
- 关闭所有打开该文件夹的程序
- 关闭文件资源管理器中打开的文件夹
- 确保没有程序正在访问文件夹内的文件

**原因2：权限不足**
- 右键以管理员身份运行程序
- 检查文件夹的读写权限

**原因3：名称包含非法字符**
- 程序会自动清理：`\ / : * ? " < > |`
- 这些字符会被替换为 `_`

**原因4：文件被锁定**
- 关闭所有可能占用文件的程序
- 杀毒软件可能锁定文件，暂时禁用

### 3. 查看详细错误日志

重命名失败时，日志会显示：
```
重命名失败: 文件夹名称, 错误: 无法重命名文件夹: XXX
源路径: F:\test\新建文件夹 - 副本 (3)
目标路径: F:\test\J76000016.3426_雾卡其_170_92A(M)_6942674037033
可能原因：文件夹被其他程序占用或权限不足
```

根据错误信息排查问题。

## 🔍 故障排查

### 问题1：剪贴板获取失败

**症状**：点击按钮后提示"剪贴板为空"或"格式不支持"

**解决**：
1. 确保在**Windows资源管理器**中选中文件夹
2. 确保按了 `Ctrl + C` 复制
3. 不要在其他程序中复制文本后再来操作

### 问题2：重命名全部失败

**症状**：成功0，失败25

**解决**：
1. 查看日志中的详细错误信息
2. 关闭所有打开的文件夹窗口
3. 关闭可能占用文件的程序（如Word、Excel等）
4. 尝试以管理员身份运行程序
5. 检查磁盘空间是否充足

### 问题3：日志显示乱码

**症状**：中文字符显示为乱码

**解决**：
1. 已修复，请重新编译运行最新版本
2. 确保系统语言支持中文
3. 尝试更换字体设置

## 💡 使用技巧

### 1. 批量处理技巧
- 每次处理20-30个文件夹为宜
- 处理前先在测试目录验证
- 确保Excel行数 ≥ 文件夹数量

### 2. 命名规范建议
- Excel中的名称要有意义
- 避免过长的名称（Windows限制260字符）
- 使用统一的命名格式

### 3. 安全操作
- 处理前备份重要数据
- 先在测试文件夹上试验
- 查看日志确认每一步操作
