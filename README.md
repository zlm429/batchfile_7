# 批量文件夹重命名工具

## 功能说明

这是一个C/S架构的桌面应用程序，用于批量重命名文件夹及其内部文件。

### 主要功能：
1. **Excel 4列数据组合**：读取Excel前4列（A+B+C+D），自动组合成文件夹名称
2. **选择性重命名**：支持鼠标多选需要重命名的文件夹，不是全部处理
3. **自动重命名文件夹内的文件**：文件夹内的所有文件会自动重命名为与文件夹相同的名字（保留原扩展名）
4. **智能冲突处理**：如果目标名称已存在，自动添加序号避免覆盖

## 技术栈

- Java 17
- JavaFX 17.0.2（图形界面）
- Apache POI 5.2.3（Excel文件处理）

## 使用方法

### 1. 准备Excel文件
- Excel文件需包含4列数据（A、B、C、D列）
- 程序会自动将4列数据用下划线连接组合成文件夹名称
- 每行对应一个文件夹名称
- 确保名称数量不少于需要重命名的文件夹数量

**Excel示例：**
```
| A列            | B列    | C列           | D列            |
|---------------|-------|--------------|---------------|
| J76000016.3426 | 雾卡其 | 170/92A(M)   | 6942674037033 |
| J76000016.3427 | 雾卡其 | 175/96A(L)   | 6942674037040 |
```

**生成的文件夹名称：**
```
J76000016.3426_雾卡其_170/92A(M)_6942674037033
J76000016.3427_雾卡其_175/96A(L)_6942674037040
```

### 2. 运行程序
```bash
mvn clean javafx:run
```

或者打包后运行：
```bash
mvn clean package
java -jar target/batchfile_app-0.0.1-SNAPSHOT.jar
```

### 3. 操作步骤
1. 点击"浏览..."按钮选择Excel文件
2. 点击"浏览..."按钮选择包含子文件夹的父目录
3. **在文件夹列表中勾选需要重命名的文件夹（可多选）**
4. 点击"开始重命名"按钮执行操作
5. 在日志区域查看操作结果

## 注意事项

⚠️ **重要提醒：**
- 此操作不可逆，请在执行前备份重要数据
- 确保Excel中的名称合法（不包含非法字符如 \ / : * ? " < > |）
- 如果目标文件夹名称已存在，系统会自动添加序号避免冲突

## 故障排除

1. 如果无法启动，请确认Java版本为17或更高
2. 如果Excel读取失败，请检查文件格式是否为.xlsx或.xls
3. 如果重命名失败，请检查文件名是否包含非法字符
4. 确保有足够的磁盘权限进行文件操作
