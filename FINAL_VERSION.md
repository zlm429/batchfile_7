## ✨ 核心功能

### 1. Excel 4列数据智能组合
- 自动读取Excel前4列（A、B、C、D）
- 用下划线(_)连接4列数据生成文件夹名称
- 支持多种数据类型（文本、数字、日期等）
- 智能处理空单元格

### 2. 选择性批量重命名
- 显示所有子文件夹列表
- 支持鼠标多选需要重命名的文件夹
- 只处理勾选的文件夹，灵活可控
- 支持Ctrl/Shift多选操作

### 3. 文件夹内文件同步重命名
- 文件夹重命名后，内部所有文件自动重命名
- 文件名与文件夹名完全一致
- 保留原文件扩展名
- 支持0个或多个文件

### 4. 智能冲突处理
- 自动检测文件名冲突
- 冲突时自动添加序号（_1, _2, _3...）
- 避免覆盖已有文件

## 📖 快速使用指南

### 步骤1：准备Excel文件

创建Excel文件，包含4列数据：

```
| A列            | B列    | C列           | D列            |
|---------------|-------|--------------|---------------|
| J76000016.3426 | 雾卡其 | 170/92A(M)   | 6942674037033 |
| J76000016.3427 | 雾卡其 | 175/96A(L)   | 6942674037040 |
| J76000016.3428 | 雾卡其 | 180/100A(XL) | 6942674037057 |
```

### 步骤2：启动程序

```powershell
# 方法1：双击运行
run.bat

# 方法2：PowerShell
.\run.ps1

# 方法3：Maven命令
.\mvnw.cmd clean javafx:run
```

### 步骤3：加载数据

1. 点击"浏览..."选择Excel文件
2. 日志显示："从Excel中加载了 X 个名称（4列组合）"

### 步骤4：选择父目录

1. 点击"浏览..."选择包含子文件夹的父目录
2. 程序自动加载所有子文件夹到列表
3. 日志显示："已加载 X 个子文件夹"

### 步骤5：勾选文件夹

在文件夹列表中勾选需要重命名的文件夹：
- **Ctrl + 点击**：选择多个不连续的文件夹
- **Shift + 点击**：选择连续范围的文件夹
- **Ctrl + A**：全选所有文件夹

### 步骤6：执行重命名

1. 点击"开始重命名"按钮
2. 查看实时日志
3. 完成后查看统计结果

##  使用示例

### 场景

您有以下文件夹：
```
D:\Products\
├── folder_001\
│   ├── photo1.jpg
│   └── photo2.png
├── folder_002\
│   └── document.pdf
├── folder_003\
│   ── (空)
└── folder_004\
    ├── data1.xlsx
    └── data2.xlsx
```

### 操作

1. 选择父目录：`D:\Products\`
2. 程序加载4个文件夹到列表
3. 勾选 `folder_001`、`folder_002`、`folder_004`（不勾选folder_003）
4. 点击"开始重命名"

### 结果

```
D:\Products\
├── J76000016.3426_雾卡其_170_92A(M)_6942674037033\
│   ├── J76000016.3426_雾卡其_170_92A(M)_6942674037033.jpg
│   └── J76000016.3426_雾卡其_170_92A(M)_6942674037033.png
├── J76000016.3427_雾卡其_175_96A(L)_6942674037040\
│   └── J76000016.3427_雾卡其_175_96A(L)_6942674037040.pdf
├── folder_003\              ← 未勾选，保持不变
│   └── (空)
└── J76000016.3428_雾卡其_180_100A(XL)_6942674037057\
    ├── J76000016.3428_雾卡其_180_100A(XL)_6942674037057.xlsx
    └── J76000016.3428_雾卡其_180_100A(XL)_6942674037057.xlsx
```

## ️ 重要提醒

### 使用前必读

1. **备份数据**
   - ️ 操作不可逆
   - 请在执行前备份重要数据
   - 建议先在测试目录上试验

2. **Excel格式**
   - 必须包含前4列（A、B、C、D）
   - 数据行数 ≥ 选中的文件夹数量
   - 不要包含非法字符：`\ / : * ? " < > |`

3. **选择文件夹**
   - 必须在列表中勾选文件夹
   - 支持多选操作
   - 只处理勾选的文件夹

4. **权限检查**
   - 确保对目标目录有读写权限
   - 关闭正在使用的文件
   - 确保文件夹未被其他程序占用
