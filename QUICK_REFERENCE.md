# 项目快速参考卡片

##  项目概述
**批量文件夹重命名工具 v2.0** - C/S架构桌面应用程序

## ✨ 核心功能
- ✅ Excel 4列数据智能组合（A+B+C+D）
- ✅ 鼠标多选文件夹（灵活选择）
- ✅ 文件夹和文件同步重命名
- ✅ 保留文件扩展名

## 🚀 快速启动
```powershell
.\run.ps1
```
或双击 `run.bat`

## 📋 使用步骤
1. 选择Excel文件（4列数据）
2. 选择父目录
3. 勾选需要重命名的文件夹
4. 点击"开始重命名"

## ️ Excel格式
```
| A列            | B列    | C列           | D列            |
|---------------|-------|--------------|---------------|
| J76000016.3426 | 雾卡其 | 170/92A(M)   | 6942674037033 |
```
生成名称：`J76000016.3426_雾卡其_170/92A(M)_6942674037033`

## ️ 重要提醒
- 操作不可逆，请提前备份
- 文件夹名不能包含：\ / : * ? " < > |
- 必须在列表中勾选文件夹
- 选中的文件夹数 ≤ Excel行数
