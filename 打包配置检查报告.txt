===========================================
️ 已知限制
===========================================

1. macOS Intel 打包
   - GitHub Actions 的 macos-latest 是 ARM64 架构
   - 打包出来的应用可能无法在 Intel Mac 上运行
   - 建议在 Intel Mac 上本地打包

2. Windows 打包
   - 使用自定义 JRE 而非 jpackage
   - 需要用户双击 start.bat 启动（无独立 EXE）
   - 包体积较大（包含完整 JRE）

3. 模块兼容性
   - 使用 Java 17 + JavaFX 17.0.2
   - Apache POI 5.2.3
   - 所有依赖版本已锁定
