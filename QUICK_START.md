# 批量文件夹重命名工具 - 快速开始指南

## 项目概述

这是一个基于JavaFX的C/S架构桌面应用程序，用于批量重命名文件夹及其内部文件。

## 功能特性

✅ 通过Excel文件提供新的文件夹名称列表  
✅ 图形界面操作，简单易用  
✅ 自动重命名文件夹内的所有文件（保留原扩展名）  
✅ 实时操作日志显示  
✅ 智能处理文件名冲突  

## 运行程序

### 方法一：使用Maven直接运行（推荐开发环境）

在项目根目录下执行：

```powershell
.\mvnw.cmd clean javafx:run
```

### 方法二：打包后运行

1. 打包项目：
```powershell
.\mvnw.cmd clean package
```

2. 运行生成的JAR文件：
```powershell
java --module-path "target\dependency" --add-modules javafx.controls,javafx.fxml -jar target\batchfile_app-0.0.1-SNAPSHOT.jar
```

## 使用步骤

### 第一步：准备Excel文件

创建一个Excel文件（.xlsx或.xls格式），在第一列（A列）中输入新的文件夹名称，每行一个名称。

示例：
```
| A列         |
|------------|
| 项目文档    |
| 客户资料    |
| 财务报表    |
| 员工信息    |
```

### 第二步：启动程序

运行上述命令启动应用程序。

### 第三步：选择Excel文件

1. 点击"浏览..."按钮
2. 选择准备好的Excel文件
3. 系统会自动加载名称列表并在日志中显示数量

### 第四步：选择要重命名的文件夹

1. 点击"浏览..."按钮
2. 选择包含需要重命名子文件夹的父目录
3. 程序会自动扫描该目录下的所有子文件夹

### 第五步：执行重命名

1. 点击"开始重命名"按钮
2. 在日志区域查看操作进度和结果
3. 完成后会显示成功和失败的数量

## 注意事项

⚠️ **重要提醒：**

1. **备份数据**：此操作不可逆，请在执行前备份重要数据
2. **名称合法性**：确保Excel中的名称不包含非法字符：`\ / : * ? " < > |`
3. **数量匹配**：Excel中的名称数量应不少于待重命名的文件夹数量
4. **权限检查**：确保对目标文件夹有读写权限
5. **名称冲突**：如果目标名称已存在，系统会自动添加序号（如：名称_1, 名称_2）

- **文件夹选择区**：选择包含子文件夹的父目录

## 技术细节

### 重命名逻辑

1. 读取Excel第一列的所有非空单元格作为新名称
2. 扫描选定目录下的所有直接子文件夹
3. 按顺序将每个子文件夹重命名为对应的新名称
4. 对于每个文件夹内的文件：
   - 提取原文件的扩展名
   - 将文件名改为与文件夹相同的名字
   - 保留原扩展名

### 示例

假设有一个文件夹结构：
```
D:\Documents\
├── folder1/
│   ├── report.docx
│   └── data.xlsx
└── folder2/
    └── image.jpg
```

Excel中的名称：
```
项目文档
客户资料
```

重命名后：
```
D:\Documents\
├── 项目文档/
│   ├── 项目文档.docx
│   └── 项目文档.xlsx
└── 客户资料/
    └── 客户资料.jpg
```

## 故障排除

### 问题1：程序无法启动
- 确认Java版本为17或更高：`java -version`
- 确认Maven Wrapper文件存在且有执行权限

### 问题2：Excel读取失败
- 检查文件格式是否为.xlsx或.xls
- 确保Excel文件未被其他程序占用
- 验证第一列包含有效的文本内容

### 问题3：重命名失败
- 检查文件名是否包含非法字符
- 确认有足够的磁盘权限
- 查看日志中的具体错误信息
- 确保目标文件夹未被其他程序占用

### 问题4：编译错误
- 清理并重新编译：`.\mvnw.cmd clean compile`
- 检查网络连接以下载依赖
- 确认pom.xml配置正确
